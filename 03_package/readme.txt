"""导包
包：作用是防止模块内容冲突
    包：有__init__.py的文件夹
    文件夹:无__init__.py的文件夹

__init__.py文件：
    作用：
        1.导包的时候默认会加载__init__.py文件
        2.调用模块使用 from pkg import * 时默认包内的所有模块都不能访问。需要在__init__.py文件中定义__all__['模块名1','模块名2']的方式向外暴露模块访问权限
        3.由于import包会加载__init__.py文件，可以在__init__.py文件里定义一些初始化的变量、函数、类。调用时可以直接通过 包名.[变变量]、[函数]、[类]访问

导包方式：
    1. import pkg  导入包(可访问__init__.py的内容)
    2. from pkg import * 导入所有包内所有__all__变量包含的模块（需要配合__init__.py文件里的__all__['模块名1','模块名2']）
    3. from pkg import utils1, utils3  按需导入包内的模块

"""
